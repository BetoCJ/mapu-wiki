# 🔌 Cómo conectarte

Conectarse a Mapucraft es muy sencillo. Sigue estos pasos:

***

## Pasos

**1.** Abre Minecraft Java Edition (versión `1.21.1`)

**2.** Ve a **Multijugador** → **Añadir servidor**

**3.** Ingresa los siguientes datos:

| Campo | Valor |
|---|---|
| **Nombre** | Mapucraft |
| **IP del servidor** | `mapucraft.com` |

**4.** Haz clic en **Listo** y luego en **Conectar**

***

{% hint style="info" %}
Si tienes problemas para conectarte, revisa que estés usando la versión correcta de Minecraft o pide ayuda en nuestro [Discord](https://discord.gg/bRjP5ruEwM).
{% endhint %}
