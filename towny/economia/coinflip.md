# 🪙 CoinFlip

El **CoinFlip** es un sistema de apuestas donde dos jugadores se enfrentan cara a cara. Cada uno pone una cantidad de dinero y la suerte decide quién se lleva todo.

***

## ¿Cómo funciona?

1. Un jugador crea una apuesta con una cantidad de dinero
2. Otro jugador acepta la apuesta
3. Se lanza la moneda — ¡el ganador se lleva el doble!

***

## Comandos

| Comando | Descripción |
|---|---|
| `/cf create [cantidad]` | Crear una apuesta |
| `/cf` | Ver las apuestas disponibles |
| `/cf join [jugador]` | Unirte a la apuesta de otro jugador |
| `/cf stats` | Ver tus estadísticas de CoinFlip |
| `/cf top` | Ver el ranking de ganancias |

***

{% hint style="danger" %}
El CoinFlip es un juego de azar. No apuestes más de lo que estás dispuesto a perder.
{% endhint %}
