# 🎮 Mini-juegos

Mapucraft cuenta con varios mini-juegos disponibles para toda la comunidad. Están en la zona de ocio accesible desde el spawn principal o mediante `/warps`.

***

## Parkour

El parkour más grande del servidor. Completa el recorrido acumulando saltos y compite en el ranking global.

| | |
|---|---|
| **Warp** | `/warp parkour` |
| **Clasificación** | Top 10 visible en el holograma del warp |
| **Récord del servidor** | Batirlo otorga 1x Llave Aleatoria de Rango |

### Recompensas por hitos

Las primeras veces que alcanzas ciertos saltos se entregan recompensas únicas (solo una vez por cuenta):

| Saltos | Recompensa |
|---|---|
| 25 | $200 + 2 MapuPoints |
| 50 | $500 + 3 MapuPoints + 25 XP |
| 75 | $750 + 5 MapuPoints + 25 XP |
| 100 | $1,250 + 7 MapuPoints + 25 niveles de XP |
| 150 | 12 MapuPoints |

Además, hay recompensas periódicas que se repiten indefinidamente:

| Cada X saltos | Recompensa |
|---|---|
| 9 | $2 |
| 20 | $4 |
| 35 | $8 |
| 80 | 1× Llave Diaria |

## Duels — 1v1

Reta a otro jugador a un duelo directo. Puedes elegir kit, arena y apostar dinero en el combate.

| | |
|---|---|
| **Warp** | `/warp preduels` |
| **Apuestas** | Soporta apuestas en dinero con `/duel [jugador] [monto]` |
| **Kits** | Selección de kits disponibles antes del duelo |
| **KeepInventory** | No pierdes tu inventario real — se usan los ítems del kit |

### Comandos

| Comando | Descripción |
|---|---|
| `/duel [jugador]` | Enviar solicitud de duelo |
| `/duel [jugador] [monto]` | Duelo con apuesta de dinero |
| `/duel accept` | Aceptar un duelo entrante |
| `/duel deny` | Rechazar un duelo |
| `/duel spectate [jugador]` | Espectar un duelo activo |
