# 🎖️ Battle Pass

El **Battle Pass** es el sistema de progresión por temporadas de Mapucraft. Completa misiones para ganar puntos, sube de nivel y desbloquea recompensas exclusivas. Existe un pase **Gratuito** (disponible para todos) y un pase **Premium** (con recompensas superiores).

***

## ¿Cómo funciona?

Cada **10 puntos** acumulados subes un nivel del pase. Los puntos se consiguen completando misiones semanales y misiones diarias. Hay un total de **98 niveles** en el pase gratuito y **100 niveles** en el premium.

Cada día se asignan **5 misiones diarias** que se renuevan a las **12:00**. Además, cada semana hay misiones semanales organizadas por semana de temporada.

El progreso de las misiones se muestra en la **barra de jefe** al alcanzar ciertos hitos (1%, 10%, 25%, 50%, 75% y 100%).

***

## Pase Gratuito vs Premium

| | Gratuito | Premium |
|---|---|---|
| **Disponibilidad** | Todos los jugadores | Requiere rango Premium |
| **Niveles** | 98 | 100 |
| **Recompensas** | Estándar | Superiores |

***

## Recompensas por tipo

A lo largo de los niveles del pase encontrarás recompensas de los siguientes tipos:

| Tipo de recompensa | Descripción |
|---|---|
| 💰 **Dinero** | Cantidades de dinero del servidor (desde $1,250 hasta $10,000+) |
| 🗝️ **Llaves** | Llaves para abrir cajas de recompensas |
| ✨ **Experiencia** | Niveles de experiencia de Minecraft |
| 💎 **Diamantes** | Diamantes en cantidad creciente |
| 🪨 **Ancient Debris** | Material para fabricar Netherite |
| ⚙️ **Netherite** | Netherite en lingotes |
| 📗 **Libros encantados** | Libros de encantamientos |
| 🟡 **Bloques de Oro** | Bloques de oro |
| 💚 **Esmeraldas** | Esmeraldas |
| 🍖 **Comida** | Alimentos del servidor |
| ⚔️ **Equipo** | Armaduras y herramientas |
| 🐾 **Spawner** | Nivel 97 (gratuito) / Nivel 99 (premium) |
| 🕊️ **Vuelo temporal** | 5 días de vuelo (nivel 98 gratuito) / 8 días (nivel 100 premium) |

***

## Misiones diarias

Las misiones diarias son tareas variadas que se renuevan cada día a las **12:00**. Se asignan **5 misiones** por día y cada una otorga puntos para el pase según su dificultad:

| Dificultad | Puntos |
|---|---|
| Fácil | 1.0 – 2.0 pts |
| Normal | 2.5 – 3.5 pts |
| Difícil | 4.0 – 5.0 pts |
| Épica | 6.0 – 10.0 pts |

Las misiones abarcan todo tipo de actividades: minar, construir, cosechar, matar mobs, pescar, encantar, fabricar ítems y más.
