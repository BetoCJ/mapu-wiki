# ✨ Encantamientos

Mapucraft cuenta con un sistema de **encantamientos personalizados** que va más allá de los encantamientos vanilla de Minecraft. Estos encantamientos ofrecen habilidades únicas que no encontrarás en otro servidor.

***

## ¿Cómo obtenerlos?

Los encantamientos custom se pueden obtener de las siguientes formas:

* 📦 **Cajas de recompensas** — Algunas cajas contienen libros de encantamientos custom
* 🏰 **Dungeons** — Los jefes dropean libros exclusivos
* 🛒 **Tiendas de jugadores** — Otros jugadores pueden venderlos

## Comandos

| Comando | Descripción |
|---|---|
| `/ae list` | Ver todos los encantamientos custom |
| `/ae info [encantamiento]` | Ver detalle de un encantamiento |
| `/enchanter` | Abre el menú para aplicar encantamientos custom a tu equipo usando libros |
| `/tinkerer` | Intercambia libros de encantamientos que no necesitas por polvo de encantamiento u otros libros |
| `/alchemist` | Combina polvo de encantamiento para obtener polvo de mayor nivel |
