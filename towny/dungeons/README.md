# 🏰 Dungeons

Los **Dungeons** de Mapucraft son mazmorras llenas de peligros y recompensas únicas. Existen tres niveles de dificultad, cada uno con sus propios mobs, jefes y botín exclusivo.

## Consejos generales

* Lleva suficiente comida y pociones
* Coordínate con tu equipo antes de entrar
* Las llaves del jefe son necesarias para acceder al cofre final
