# 💼 Jobs

El sistema de **Jobs** (empleos) te permite ganar dinero realizando distintas actividades en el servidor. Elige el trabajo que más se adapte a tu estilo de juego.

***

## Trabajos disponibles

| Trabajo | Actividad principal |
|---|---|
| ⛏️ **Minero** | Minar bloques |
| 🌾 **Agricultor** | Cultivar y cosechar |
| 🪵 **Leñador** | Cortar árboles |
| 🏹 **Cazador** | Matar mobs |
| 🎣 **Pescador** | Pescar |
| 🔨 **Constructor** | Colocar bloques |
| ⚗️ **Alquimista** | Elaborar pociones |
| ⚔️ **Soldado** | Combatir jugadores |

***

## Comandos

| Comando | Descripción |
|---|---|
| `/jobs browse` | Ver todos los trabajos disponibles |
| `/jobs join [trabajo]` | Unirte a un trabajo |
| `/jobs leave [trabajo]` | Abandonar un trabajo |
| `/jobs info [trabajo]` | Ver información de un trabajo |
| `/jobs stats` | Ver tus estadísticas |

***

{% hint style="info" %}
Puedes tener hasta **[N]** trabajos activos al mismo tiempo. Completa esta información según tu configuración.
{% endhint %}
