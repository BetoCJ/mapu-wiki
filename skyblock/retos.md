# 🎯 Retos del Wenu Mapu

Cada 10 minutos, el servidor lanza un **reto aleatorio** para todos los jugadores conectados: el primero (o los primeros 3) en completarlo recibe una recompensa en monedas.

***

## Cómo funciona

- Un nuevo reto aparece automáticamente cada **600 segundos** (10 minutos).
- Tienes **120 segundos** para completarlo antes de que expire.
- Se anuncia con el prefijo **"RETOS DEL WENU MAPU"** en el chat.
- Los primeros 3 jugadores en completarlo entran al ranking del reto.

## Tipos de retos

| Categoría | Ejemplo |
|---|---|
| **Consumir** | Comer cierta cantidad de un alimento (ej. Bistec, Manzana Dorada) |
| **Romper bloques** | Picar X cantidad de un bloque (ej. Piedra, Mineral de Diamante) |
| **Colocar bloques** | Colocar X cantidad de un bloque |
| **Soltar ítems** | Soltar X cantidad de un ítem |
| **Pescar** | Pescar X peces de un tipo |
| **Craftear** | Craftear X cantidad de un ítem |
| **Fundir** | Cocinar/fundir X cantidad de un ítem en el horno |
| **Matar** | Eliminar X cantidad de un tipo de mob |
| **Trivia** | Responder correctamente una pregunta sobre el servidor o cultura Mapuche |
| **Mini-retos** | Acciones especiales (saltar, tocar una campana, comerciar con un aldeano) |
| **Daño** | Recibir daño de un tipo específico (caída, fuego, ahogamiento) |

## Trivia

Existen alrededor de **90 preguntas** de trivia sobre el lore Mapuche del servidor y curiosidades generales, por ejemplo:

> ¿Cómo se llama la serpiente protectora de la tierra que elevó los cerros?
> *(Respuesta: Treng Treng Vilu)*

> ¿Cuál es el primer rango de progresión para los jugadores nuevos?
> *(Respuesta: Weche)*

***

{% hint style="info" %}
Los retos ocurren en el mundo principal del servidor (incluyendo Nether y End), así que puedes encontrarte uno mientras exploras o esperas en el hub.
{% endhint %}
