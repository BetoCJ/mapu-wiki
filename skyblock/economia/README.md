# 💰 Economía

La economía del servidor combina una **tienda oficial**, **tiendas de jugadores**, una **casa de subastas** y comercio directo entre jugadores.

***

## 🏪 Tienda oficial

```
/tienda
```

La tienda oficial está organizada en categorías como **bloques**, **comida**, **minerales**, **redstone**, **tintes**, **herramientas**, **armaduras**, **drops de mobs**, **decoración** y **minions**. Cada categoría tiene su propia interfaz de compra y venta.

Algunas categorías usan monedas especiales en vez de dinero normal — por ejemplo, las armaduras elementales se compran con materiales temáticos como **Chaway de Plata**, **Hilo de Relmu** o **Fibra de Voqui**, obtenidos jugando.

## 🔨 Subasta — zAuctionHouse

```
/ah
```

| Comando | Descripción |
|---|---|
| `/ah sell [precio] [cantidad]` | Publicar un ítem en venta |
| `/ah` | Ver los ítems en venta |
| `/ah search [texto]` | Buscar un ítem |
| `/ah claim` | Reclamar el dinero de tus ventas |
| `/ah expired` | Ver tus ítems caducados |
| `/ah purchased` | Ver tu historial de compras |
| `/ah history` | Ver tu historial de ventas |

Cada publicación dura **2 días** antes de caducar; si no se vende, el ítem vuelve a tu inventario de objetos caducados para que lo reclames.

## 🤝 Intercambio directo — AxTrade

```
/trade [jugador]
```

Abre una interfaz de intercambio seguro con confirmación de ambas partes. Debes estar a un máximo de **10 bloques** del otro jugador, y el trade se cancela automáticamente si alguno se mueve, interactúa con bloques o ejecuta comandos durante la negociación.
