# 🎖️ Battle Pass — El Ciclo del Küyen

Cada temporada trae **El Ciclo del Küyen**, el pase de batalla del servidor con temática de fases lunares.

```
/battlepass
```

***

## ⚙️ Cómo funciona

- El pase tiene **50 fases** (tiers).
- Cada fase cuesta **1,250 XP de Battle Pass**, que se gana completando tareas y misiones diarias/semanales.
- Existen dos sendas de recompensas:
  - **Senda Libre** — disponible para todos los jugadores.
  - **Senda Astral (Premium)** — requiere haber adquirido el pase premium.

## 🎁 Ejemplos de recompensas por fase

| Fase | Recompensa Gratuita | Recompensa Premium |
|---|---|---|
| 1 | 500 monedas | 1,000 monedas |
| 3 | 32 Bistecs | 64 Bistecs |
| 5 | Llave Küyen | Llave Antü |
| 7 | Espada de Hierro (Sharpness II) | Espada de Diamante (Sharpness III) |
| 11 | 5,000 monedas | 10,000 monedas |
| 20 | Pico de Diamante (Efficiency III) | Pico de Netherite (Efficiency IV + Unbreaking III) |
| 25 | Llave Küyen | Llave Pillán |
| 35 | 50,000 monedas | 100,000 monedas |

A lo largo del pase también se reparten **Minions** (Minero, Granjero, Cazador, Recolector), libros de encantamiento, totems y lingotes de netherita.
